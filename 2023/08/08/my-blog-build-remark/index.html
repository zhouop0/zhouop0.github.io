从零搭建github博客

采用 Github Pages + Hexo 的方式，搭建个人博客。

一.准备工作

1. 使用个人 GitHub 创建仓库，并配置 GitHub Pages

注意: 此仓库用于存放个人博客页面，仓库名必须使用 .github.io格式。

1 npm install -g hexo-cli
2 hexo -v # 查看版本，目前最新版本为4.3.1

1 hexo init my-blog
2 cd my-blog
3 npm install

1 hexo generate # 生成页面，此命令可以简写为 hexo g
2 hexo server # 本地启动，可简写为 hexo s

1 cd themes # 此目录为my-blog代码目录中themes目录
2 git clone https://github.com/Siricee/hexo-theme-Chic.git chic

theme: chic

hexo s


启动我们的博客进行预览

可以看到修改成功。
